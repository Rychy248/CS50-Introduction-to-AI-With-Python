Elements

  1. Agent: Represent the entity into the context, example into a search for shorter way to get from a to b point, the agent could be the car.
  2. State: It's the initial situation of the context and the agent.
  3. Action: Actions(S) returns the set of actions that can be executed in state S, an action can be set under a certain state and it cannot be set under other. Example: if we have the fifth numbers to order, we only can move up, right, left or bottom the square of one number not of two or n where n!=1
  4. Transition model: This describe the relation between the state and the Action, this describe how the state react or act by under action as a param. Results(s, a) returns the state resulting from performing action a in state s
